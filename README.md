# О проекте
Версия интерпретатора Python 3.12.9

# Зависимости
Проект использует библиотеки math и random, включённые в стандартную библиотеку Python

# Как запустить
Открыть командную строку, перейти в папку проекта, выполнить команду python main.py. Убедитесь, что в системе установлен совместимый интерпретатор Python и добавлен в PATH
